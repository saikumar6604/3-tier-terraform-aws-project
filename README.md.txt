# 🚀 3-Tier Web Application Deployment using Terraform (AWS)

## 📌 Overview

This project demonstrates deployment of infrastructure on AWS using Terraform.

## 🏗️ Architecture

* EC2 instance (Application layer)
* Load balancer (conceptual)
* Database layer (conceptual)

## ⚙️ Features

* Infrastructure as Code
* Automated provisioning

## 🛠️ Tech Stack

* AWS (EC2)
* Terraform

## 🚀 How to Run

terraform init
terraform apply
